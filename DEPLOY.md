# デプロイ手順

このゲームは静的ファイルのみで構成されているため、様々なホスティングサービスに簡単にデプロイできます。

## 推奨デプロイ方法

### 1. GitHub Pages（無料・簡単）

1. GitHubにリポジトリを作成
2. Settings > Pages に移動
3. Source を "main" ブランチの "/ (root)" に設定
4. 数分で `https://[ユーザー名].github.io/[リポジトリ名]` で公開されます

### 2. Netlify（無料・ドラッグ&ドロップ対応）

1. [Netlify](https://www.netlify.com/) にアクセス
2. サイトをドラッグ&ドロップでアップロード
3. 即座に公開URLが発行されます

### 3. Vercel（無料・高速）

1. [Vercel](https://vercel.com/) にアクセス
2. GitHubリポジトリを接続
3. 自動的にデプロイされます

## デプロイ前の確認事項

- [ ] すべての画像ファイルが `public/` フォルダに正しく配置されている
- [ ] ローカルサーバーで動作確認済み
- [ ] ブラウザのコンソールにエラーがない
- [ ] モバイル表示も確認済み（必要に応じて）

## 開発環境と本番環境の違い

このゲームは完全にクライアントサイドで動作するため、開発環境と本番環境で動作に違いはありません。

ただし、以下の点に注意：
- ローカルサーバーで開発することで、本番環境に近い状態でテストできます
- 相対パス（`public/` など）が正しく動作することを確認してください
